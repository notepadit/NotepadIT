PC C:\Users\it-admin> WSTĘPNA KONFIGURACJA SYSTEMU WINDOWS SERVER

Utworzenie folderów docelowych dla plików VM oraz VHDx:

Komenda 1: New-Item E:\VM -ItemType Directory

Komenda 2: New-Item E:\VHD -ItemType Directory

Instalacja roli serwera Hyper-V (wymagany restart):

Komenda 3: Install-WindowsFeature Hyper-V -IncludeManagementTools -Restart

Instalacja narzędzi do zdalnego zarządzania serwerem Hyper-V:

Komenda 4: Install-WindowsFeature RSAT-Hyper-V-Tools

Konfiguracja serwera Hyper-V do używania folderów docelowych VM/VHDx:

Komenda 5: Set-VMHost -ComputerName HV01 -VirtualHardDiskPath E:\vhd -VirtualMachinePath E:\vm

Potwierdzenie poprawności konfiguracji:

Komenda 6: Get-VMHost -ComputerName HV01 | Format-List *
