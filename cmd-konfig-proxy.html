C:\Users\it-admin> KONFIGURACJA USTAWIEŃ SERWERA PROXY

Poniżej komendy umożliwiające zmianę/naprawę globalnych ustawień serwera proxy w systemie Windows:

Komenda 1: netsh winhttp show proxy
Komentarz: Sprawdzenie aktualnych ustawień serwera proxy

Komenda 2: netsh winhttp import proxy source =ie
Komentarz: Zaimportowanie ustawień serwera proxy z "Opcji internetowych"

Komenda 3: netsh winhttp set proxy proxy-server=192.168.1.1:3128
Komentarz: Ręczna konfiguracja adresu serwera proxy

Komenda 4: netsh winhttp reset proxy
Komentarz: Usunięcie ustawień serwera proxy
